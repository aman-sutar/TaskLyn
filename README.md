# Tasklyn – Personal Task Manager

Tasklyn is a personal task management web application designed to help you organize tasks, track productivity, log activity, and manage notes in one place.

## Features

### Dashboard

* Overview of total tasks, completed tasks, tasks in progress, and focus time
* Today’s tasks section
* Task progress overview with completion percentages
* Weekly activity summary (focus and break time)
* Quick notes preview

### Tasks Management

* Add, edit, and delete tasks
* Task details include title, description, priority, color, status, due date, and progress
* Kanban-style board with:

  * To Do
  * In Progress
  * Completed
* Filter tasks by status and priority
* Clear completed tasks

### Activity Tracking

* Log daily focus time and break time
* Select specific days of the week
* Weekly activity overview with charts
* Summary of total focus, total break, and daily average focus time

### Notes

* Create, edit, and view personal notes
* Color-themed notes
* Quick notes section on the dashboard
* Dedicated notes view for all notes

### Analytics

* Total tasks count
* Completed tasks count
* Notes count
* Day streak
* Task distribution chart
* Weekly focus trend chart

### Profile & Settings

* User profile with name, email, and avatar selection
* Avatar picker
* Reset all stored data

### Search & Notifications

* Search tasks and notes
* Notification panel with clear option

### Responsive Design

* Desktop navigation bar
* Mobile bottom navigation for quick access

## Modals Included

* Task modal
* Note modal
* Profile settings modal
* Search modal

Tasklyn is built as a single-page interface with multiple views, offering a clean and organized way to manage daily productivity.
